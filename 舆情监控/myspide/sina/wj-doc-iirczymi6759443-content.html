\u3000\u3000原标题：5G潮尚未来到 价格战已然升级

', '

\u3000\u3000本报记者\xa0赵鹏\xa0赵莹莹

', '

\u3000\u3000伴随着疫情好转，手机产业链迎来“满血复活”，但残酷的价格战也呼啸而来。

', '

\u3000\u30004月15日，起价2999元的荣耀30面世，当晚3299元的iPhone\xa0SE在没有任何预热的情况下突然入市。引人注目的是，新款iPhone\xa0SE并非一台鸡肋的低配手机，甚至被部分网友戏称为“卖芯片送手机”。时隔四年，苹果再次回到3000元的价格区间，无疑将对势头正猛的国产中档手机造成很大压力，也显示出4G向5G过渡之际，手机市场的洗牌力度空前提升。

', '

\u3000\u3000苹果施压国产手机

', '

\u3000\u3000疫情全球蔓延之际，苹果重新放低了身段，推出3299元起的国行版iPhone\xa0SE。4.7英寸的小屏幕手机，配备1200万像素广角摄像头，支持快速充电，还搭配了iPhone迄今最快的芯片――A13仿生芯片。显然，在5G手机“缺位”的状况下，苹果想用一款便宜的iPhone来巩固市场，以应对全球经济陷入低迷可能造成的高端消费市场损失。

', '

\u3000\u3000除了应对疫情的考量，放低身段也是苹果扭转市场份额下滑的重要举措。数据显示，从2015年到2019年，苹果手机出货量下降了近15%，而同期全球智能手机出货量从12.92亿台增长到14.86亿台，整体增加了15%。2019年，苹果被华为反超，拱手让出全球第二大手机厂商的位子。

', '

\u3000\u3000有了iPhone\xa0SE后，苹果3000元到4000元的空档已被填补，这也是中低端4G手机最主要的价格区间。

', '

\u3000\u3000如今，手机用户正变得更挑剔，既渴望新技术加持，又需要支付、云会议、文档处理、游戏等各种主流应用场景的体验优化。在此之前，金立、360、黑莓、HTC、联想等品牌的手机已陆续淡出国内市场。而随着苹果、华为两大巨头同时加码中端市场，规模较小的手机厂商生产空间更趋狭窄。

', '

\u3000\u3000手机市场疯狂推新

', '

\u3000\u30003月份以来，国内手机厂商纷纷加大新机型投放，陆续上市的机型已达40多款。

', '

\u3000\u3000继4月15日华为发布荣耀30系列手机后，4月16日，一向主打欧美高端市场的一加也在国内发售了一加8系列入市搅局。售价5999元，一加8\xa0Pro搭载了高通骁龙865的双模5G芯片，还采用了全球最高端的6.78寸120赫兹显示屏，具有更顺畅的过渡和触摸响应能力。

', '

\u3000\u3000事实上，今年2月中旬国内首款5G旗舰机小米10的推出，就拉开了国产手机推新的浪潮，近期更是成为有史以来新品发布最密集的时段。

', '

\u3000\u3000伴随着各大厂商集体发力，消费者的信心也在迅速恢复。中国信通院最新数据显示，3月份国内手机销量迅速回升，当期4G手机出货1484.4万部，5G手机则为621.5万部。而今年2月4G手机出货量仅为396万部，5G手机更是只有区区238万部。

', '

\u3000\u3000面对密集登场的各款5G手机，市民的荷包却有些吃紧。

', '

\u3000\u3000自售价5999元的国内首款5G高端旗舰机小米10面世后，售价6999元的OPPO\xa0Find\xa0X2，还有8888元的华为P40\xa0Pro+，2至4月间陆续推出的5G旗舰手机价格都让消费者有些吃惊。毕竟，5G爆款应用的缺乏，使得在实际使用中5G手机相比4G手机的优势并不明显。

', '

\u3000\u3000中低端芯片成5G普及关键

', '

\u3000\u3000今年作为5G普及的重要一年，相比价格高企的5G高端手机，公众期待更多5G手机进入2000元的亲民档位。

', '

\u3000\u3000虽然深藏不露，但中低端芯片已经开始扮演5G推手的角色。

', '

\u3000\u3000采用了高通顶级芯片骁龙865的小米10卖到了5999元，而采用了骁龙765G的Redmi\xa0K30\xa05G的价格则下探到了1698元，成为目前国内售价最低的5G手机。

', '

\u3000\u3000为了加速普及5G手机，华为3月30日发售了低端5G芯片麒麟820。这一举措立竿见效，首款搭载了麒麟820的荣耀30S最低价只有2399元。

', '

\u3000\u3000伴随着4月15日面世的荣耀30系列手机，华为也同步首发了全新麒麟985这款5G芯片。在这一中端芯片的加持下，荣耀30系列起价也只有2999元。

', '

\u3000\u3000在高通与华为之外，紫光和联发科也在发力。有消息称，即将推出的魅族17G将采用联发科5G芯片天玑1000，起售价很可能是1998元。而海信也将发售首款5G手机F50，这将是紫光展锐“中国芯”虎贲T7510的首秀，预计将让海信5G手机的价格处于2000元档位。

', '
