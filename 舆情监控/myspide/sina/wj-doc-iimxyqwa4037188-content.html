', '

\u3000\u30003月28日19时，四川凉山木里县发生森林火灾，超过2000人参加扑救。

', '

\u3000\u3000长安街知事注意到，一年前，当地曾发生原始森林遭雷击引发的火灾，致31人遇难，其中包括27名消防员。

', '

\u3000\u3000火灾地点位于乔瓦镇锄头湾村与项脚乡项脚村交界处，当地多为云南松、杂灌，过火面积目测约15公顷。

', '

\u3000\u3000按照制定方案，扑火力量1330人进入火场，采取砍设隔离带和直接扑打的方式进行扑救。由于地形复杂，29日15时许前指决定，扑救人员撤离至附近安全区域休整，并留守足够人员观察火情。

', '

\u3000\u300029日16时许，由于气温高，突起大风，火烧迹地飞火引燃火场东北线、北线外约2km处，火场被突破。

', '

\u3000\u3000今天，当地将组织扑救力量2182人参与扑救。阿坝、甘孜森林消防已调集队伍，做好增援准备。

', '

\u3000\u3000去年3月30日17时，木里县境内因雷击发生森林火灾。火场平均海拔4000米，多个火点位于悬崖峭壁上，地形复杂、坡陡谷深，交通、通讯不便。

', '

\u3000\u3000当时共689人在原始森林展开扑救。扑火行动中，受风力风向突变影响，突发林火爆燃，瞬间形成巨大火球，在现场的扑火人员紧急避险，但27名森林消防员和3名地方扑火人员失联，后证实遇难。

', '

\u3000\u3000此外，木里林业局第三营造管护处副主任王慧蓉，扑火返回途中从山坡滑落，失血性休克死亡，成为为救火而捐躯的第31名勇士。

', '
