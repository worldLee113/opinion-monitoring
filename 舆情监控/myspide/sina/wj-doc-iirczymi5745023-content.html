\u3000\u3000原标题：黑龙江矿砂泄漏事件后续：呼兰河钼浓度全线达标

', '

\u3000\u3000据央视新闻消息，2020年3月28日13时40分，中铁集团伊春鹿鸣矿业有限公司钼矿尾矿库4号溢流井发生倾斜，导致泄水量增多并伴有尾矿砂，对水环境造成一定污染。

', '

\u3000\u30004月11日黑龙江省政府新闻办公室在哈尔滨举行新闻发布会，对处置的措施和进展情况进行介绍。

', '

\u3000\u3000呼兰河钼浓度全线达标

', '

\u3000\u3000发布会上，黑龙江省政府副秘书长李明春介绍，经过14天的昼夜奋战，尾矿砂泄漏的特征污染物钼已得到有效控制，依吉密河钼浓度总体大幅下降，但受清污工程搅动影响微量超标，根据今日（4月11日）凌晨3时的监测数据显示，呼兰河钼浓度全线达标。

', '

\u3000\u3000李明春表示，“不让超标污水进入松花江”的应急处置目标能够实现。李明春还表示，下一步，将继续监测河流水质，持续关注流域环境质量情况。全面开展环境事件调查，结果及时向社会公开，同步开展生态环境损害评估，统筹谋划生态修复治理工作，并依法依规严肃追责问责。

', '

\u3000\u3000环境应急专家：不会影响居民的饮用水安全

', '

\u3000\u3000发布会上，国家环境应急专家组专家成员、哈尔滨工业大学教授马放介绍，伊春鹿鸣矿业尾矿泄漏的泥浆液中主要含有钼、矿渣、有机物等。因此，依据可能带来的水环境影响，本次环境污染事件导致水体污染的特征污染物确定为：钼、油和COD（化学需氧量）。

', '

\u3000\u3000马放还表示，钼是生物必须的微量元素，少则有益，多则有害。就本次事件来看，主要是针对特征性污染物钼，以絮凝沉淀技术为核心构建了应急技术体系。根据大量的监测数据表明，应急处理后的水中，钼的含量小于0.07mg/L，符合饮用水的标准，不会影响居民的饮用水安全。

', '

\u3000\u3000伊春、绥化不误农时备春耕

', '

\u3000\u3000眼下正值备春耕生产阶段，黑龙江省伊春、绥化的一些农田受到了本次泄漏影响。

', '

\u3000\u3000发布上伊春市人民政府副秘书长闫立海表示，伊春将全面抓好受过水影响农户备春耕生产，逐户逐地开展精准排查，采取有效措施，抢抓农时、不误春耕，最大限度减少农业生产损失。切实加强安全生产监管，统筹推进河道、林地、农田修复治理，全力把环境影响程度降到最低。

', '

\u3000\u3000绥化市政府副市长王雪峰表示，呼兰河流域自流灌区是绥化优质水稻区，初步统计，涉及水田面积43.5万亩。对于春耕生产问题，绥化市高度重视，为确保农民泡田不受影响，政府积极与专家研究对策，决定分区分类施策，采取傍河取水、打补水井等多种措施解决水稻用水问题，确保粮食产量不受影响。

', '

\u3000\u3000环境应急专家：絮凝剂对水环境有没有影响

', '

\u3000\u3000发布会上，国家环境应急专家组组长、中国环境科学研究院副院长宋永会介绍，污染的尾矿颗粒十分细，粒度在200目以上（直径75微米以下），悬浮于水中不易沉降，造成水的浊度很高和钼污染。为了去除污染物，经研究，采取投加絮凝剂沉淀悬浮物和钼污染物的应急技术路线，筛选的絮凝剂聚丙烯酰胺和聚合硫酸铁都是饮用水处理使用的药剂，不会对水环境造成影响。
