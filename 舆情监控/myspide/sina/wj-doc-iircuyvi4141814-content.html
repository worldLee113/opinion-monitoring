\u3000\u3000原标题：湖北：1-4月全省主要经济指标明显改善，物价涨幅回落

', '

\u3000\u30005月19日，湖北省统计局官网刊文介绍了今年1-4月湖北经济运行情况。总体上看，随着复工复产复市稳步推进，1-4月，全省主要经济指标明显改善。工业、投资、消费稳步恢复，财政、进出口降幅收窄，金融总体稳定，物价涨幅回落。

', '

\u3000\u3000其中，1-4月地方一般公共预算收入完成758.42亿元，下降43.7%，降幅比一季度收窄3.9个百分点。其中税收收入608.82亿元，下降42.4%，降幅收窄3.5个百分点。地方一般公共预算支出2255.33亿元，下降5.4%，降幅收窄8.7个百分点。

', '

\u3000\u30001-4月，全省规上工业增加值同比下降33.6%，降幅比一季度收窄12.2个百分点。其中，4月当月同比下降2.4%，降幅比上月收窄44.5个百分点。41个工业行业大类中，15个行业当月增加值实现正增长。

', '

\u3000\u3000大中型企业降幅相对较小，小微型企业恢复加快。数据显示，1-4月，全省规上工业大中型、小微企业增加值同比分别下降26.9%、42.1%，降幅比一季度分别收窄10.6个、14.5个百分点。

', '

\u3000\u3000另外，1-4月，全省规上工业出口交货值下降26.5%，降幅比一季度收窄18.2个百分点。工业用电量同比下降23.7%，降幅收窄7.2个百分点。

', '

\u3000\u3000消费市场方面，1-4月，全省共实现社会消费品零售总额4118.54亿元，同比下降41.0%，降幅比一季度收窄3.9个百分点。其中4月当月同比下降28.4%，降幅比上月收窄20.7个百分点。

', '

\u3000\u3000另据湖北调查总队调查，1-4月，全省居民消费价格上涨5.7%，涨幅比一季度回落0.4个百分点。全省工业生产者出厂价格由一季度的同期持平转为下降0.4%。全省工业生产者购进价格由一季度的上涨0.5%转为下降0.4%。

', '
