\u3000\u3000原标题：工信部：外贸中小企业遇到的困难比较大，正在密切关注

', '

\u3000\u30002020年5月20日，工业和信息化部部长苗圩在国务院新闻发布会上提到，经济全面的复苏还需要时间，当前中小企业发展仍然面临着复杂和严峻的局面。随着疫情在全球加速蔓延，除了防疫有一些特别的行业受到影响以外，最近一个特别突出的问题就是外贸企业，特别是外贸的中小企业遇到的困难比较大。

', '

\u3000\u3000苗圩表示，这些都是新情况、新问题，工信部还在密切关注，也准备针对不同的特殊行业出台更有针对性的帮助中小企业的各项政策，来支持广大的中小企业渡过眼前的难关。

', '

\u3000\u3000发布会上，有记者问到，疫情对中小微企业的影响比较大，我国在帮助中小微企业纾难解困方面采取哪些措施？

', '

\u3000\u3000对此，苗圩表示，这次疫情对所有的企业都带来了不利的影响，但是对中小微企业的影响要远远大于其他大型企业。各部门各地方及时出手，打出了一套帮助中小微企业政策的“组合拳”。

', '

\u3000\u3000归纳起来大概有几个方面：

', '

\u3000\u3000第一，加大减税降费，降低企业负担。加大对小微企业和个体工商户增值税的减免，对受疫情影响较大的交通运输、餐饮、旅游等行业的中小企业，亏损结转的年限由过去的5年延长到8年，还减免了各类企业上半年3项社保费，光这一项就达到了6千亿元，受益最大的是中小微企业。交通运输部门还免收了收费的公路通行费，一共达到了1400多亿元。我们的要素保障部门，降低电价、气价，上半年为企业减负达到670亿元。

', '

\u3000\u3000第二，加大金融支持，保证中小企业不至于因为现金流的中断而走入困境。人民银行实施了降准，释放出1.75万亿元的资金，通过专项再贷款、再贴现等方式，为中小微企业和个体工商户提供了低成本的贷款达到2.85万亿元，对超过100万户的中小微企业超过1万亿元的贷款本息办理了延期还本付息，这些都是支持中小企业金融方面的政策。

', '

\u3000\u3000第三，优化政府公共服务，推动企业创业创新。在疫情期间，我们开展了“企业微课”线上培训，为中小企业免费的送政策、送技术、送管理。同时，实施了中小企业数字化赋能的专项行动，帮助中小企业努力的通过数字技术的应用来降成本、拓市场、稳就业。疫情以来我们还引导各地中小企业的服务机构，累计举办各种为中小企业服务的活动达到26000多场次，开展了一对一服务达到230多万次，服务的中小企业接近593万家次。

', '

', '

\u3000\u3000苗圩称，下一步，还将继续发挥好国务院促进中小企业发展工作领导小组办公室的作用，坚持市场化的方向，会同有关部门针对中小企业存在的问题，加强分类指导，精准帮扶。一方面，要狠抓各项惠企政策措施的落地落实，进一步加大金融对中小企业的支持，加大清欠力度。同时规范各地涉及中小企业的各项收费，让广大中小企业获得实实在在的获得感。另一方面，还要不断培育中小企业生存和发展的内生性的活力和动力，因为外因是变化的条件，内因才是变化的根本，引导中小企业专注于细分领域，努力的走专精特新的发展道路，能够术业专攻，集中所擅长的实业和行业，能够扬长避短，克服自身的不利条件，发挥好自身的作用。
