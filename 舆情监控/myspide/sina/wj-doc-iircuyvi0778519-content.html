\u3000\u3000原标题：12年来最长五一小长假开启！看完这七条再往外冲

', '

\u3000\u3000来源：国是直通车

', '

\u3000\u3000今年的五一假期由三天延长至五天，是12年来首次，也是中国疫情防控进入常态化后的第一个旅游长假，不少憋闷已久的民众纷纷摩拳擦掌，准备利用假期好好“放风”一下。

', '

\u3000\u3000面对高涨的出游热情，为确保愉快安全过节，防止疫情反弹，五一前夕，4月30日下午5时，国务院新闻办举行发布会，针对即将到来五一假期发布若干旅游提示。

', '

\u3000\u3000提示一：

', '

\u3000\u3000旅游景区实行限量开放

', '

\u3000\u3000文化和旅游部部长雒树刚表示，五一期间旅游资源是限量开放，还不能完全开放，这个限量是两个方面：一是资源的限量，比如相当一部分景区的室内部分暂时不能开放，这是疫情防控的要求。二是人数的限量，旅游景区接待人数不能超过景区最大承载量的30%。

', '

\u3000\u3000提示二：

', '

\u3000\u3000旅游景区实行预约制度

', '

\u3000\u3000雒树刚表示，五一期间旅游景区将推行预约制度，实行预约开放。各地旅游景区现在已通过实施分时段的预约制度，引导游客间隔入园、错峰旅游，避免游客出行时间过于集中。游客行前要评估游览的可行性，应提前做好功课，主动学习旅游景区的疫情防控宣传引导知识。没有预约上的游客，千万不要自行前往。

', '

\u3000\u3000提示三：

', '

\u3000\u3000游客须自觉遵守防疫措施

', '

\u3000\u3000雒树刚表示，各旅游景区将对景区入口、核心景点、交通接驳点、狭窄路段等关键区域，根据瞬时承载量制定切实可行、灵活机动的流量管控措施。同时将采取提前开园时间、检查点前置、开放多个入口等方式，避免游客在景区入口区域形成聚集。对重要游览点、餐饮休息等容易形成游客拥堵的区域要采取有针对性的措施，保持人员间距。

', '

\u3000\u3000雒树刚表示，五一期间各个景区都有一些疫情防控的措施，这些措施可能会给游客出游增加一些环节，希望游客主动配合有关部门和管理者采取必要的疫情防控和应急处置措施，自觉遵守公共秩序，自觉与其他游客保持间距。

', '

\u3000\u3000提示四：

', '

\u3000\u3000气温升高病毒传播能力不一定减弱

', '

\u3000\u3000对于五一期间多地气温升高是否会减弱病毒传播能力，中国科学院应对疫情科研攻关组组长、中国科学院前沿局局长徐涛院士表示，目前普遍认为新型冠状病毒主要通过飞沫、直接接触等方式传播。至于新冠病毒是否受气温的影响，目前还处于认知当中，还需要更多数据来说明。目前最重要的还是要做好个人防护，尽量不要到高风险地区，做到勤洗手、戴口罩、常通风、少聚集，这样才能最大可能降低传播风险。

', '

\u3000\u3000提示五：

', '

\u3000\u3000不适及时告知工作人员

', '

\u3000\u3000雒树刚表示，民众出行需要严格落实体温筛检等防控措施，结合实际，配套使用“健康码”核验等手段加强筛查。在参观游览、接受游览服务时，应当向旅游景区如实告知自己个人的健康信息，遵守旅游活动中的安全警示规定，而且要做好个人防护，配合旅游景区测量体温，如果到了景区感觉到身体不太舒服，出现了不适情况，要第一时间告诉景区工作人员。

', '

\u3000\u3000提示六：

', '

\u3000\u3000老年体弱者谨慎出行

', '

\u3000\u3000中国工程院副院长、中国医学科学院院长、北京协和医学院校长王辰院士表示，老年人是易感人群，抵抗力更低，更容易被传染，而且一旦传染上，就特别容易形成重症，要加强防护。如果对青壮年旅游相对放开一些，对老年人而言，就更加要有警戒之心，身体弱的、年龄大的不一定赶着在疫情阶段或者潜在疫情阶段去旅游。对老年人和孩子要注重防护，五一是否出游，要多一番考虑。

', '

\u3000\u3000雒树刚表示，对老年人和儿童，各地旅游景区都设置了专门接待服务人员，老人和儿童在旅游过程中如出现不适状况，需要景区协助的，可以向景区及时主动提出。现在很多景区都有APP，游客到景区门口可以先下载，有急事联系，工作人员会及时提供帮助。

', '

\u3000\u3000提示七：

', '

\u3000\u3000做到文明旅游

', '

\u3000\u3000雒树刚表示，希望游客高度重视旅游文明行为，要自觉做文明旅游的践行者，要爱惜生态环境，不随地乱扔垃圾，要保护文物古迹，不乱刻乱划，要维护文明秩序。旅游餐饮单位要加强餐桌文明的宣传，就餐倡导使用公勺、公筷。

', '

\u3000\u3000王辰表示，旅游期间还要注意咳嗽、打喷嚏的文明礼仪，在咳嗽、咳痰时，用餐巾纸捂住口鼻，手不要接触用过的餐巾纸里面，团好后处理掉，或者注意用肘弯遮挡一下，避免病毒通过飞沫传播。

', '

\u3000\u3000“五一期间，要放松心情，但是不能放松安全防范。”雒树刚表示，出游期间一定要做好自我防护，包括疫情的防控和人身安全防护，要做到几个“不去”：人聚集人多的时候不去；不安全的地方不去；没有开放的地方不去；不具备旅游接待条件的地方不去。（陈溯）
