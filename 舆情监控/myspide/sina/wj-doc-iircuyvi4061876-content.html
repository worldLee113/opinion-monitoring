\u3000\u3000原标题：5月“降息”按下暂停键，未来降息过程仍会持续推进

', '

\u3000\u3000新京报讯（记者 程维妙）5月20日，全国银行间同业拆借中心公布5月贷款市场报价利率（LPR），1年期LPR为3.85%，5年期以上LPR为4.65%，均与上月持平。

', '

\u3000\u3000市场对此已有预期。为银行补充短期资金的逆回购操作已连续一个半月没有“开张”，补充长期中长期资金的MLF（中期借贷便利）操作在上周缩量平价续作。民生银行首席研究员温彬认为，5月LPR大概率会随MLF利率一样不做调整。他称，在全国两会召开前，货币政策会保持相对稳定，主要还是进一步观察，看政府工作报告对今年工作目标的具体设定。

', '

\u3000\u3000今年LPR在2月、4月已两度降息，1年期LPR年内累计下行30个基点，5年期以上LPR累计下行15个基点。对个人层面，LPR降息能给房贷一族减负。由于多数房贷期限超过5年，参照5年期以上LPR，100万贷款本金、30年等额本息的按揭贷款，年内两次降息后月供累计可省约90元。

', '

\u3000\u3000近日市场机构报告测算的“全国房贷利率连降5个月”，也是基于LPR的下降。值得一提的是，各地在2019年10月新增房贷利率正式挂钩LPR时，曾提出不同的加减点要求：如中山、无锡、苏州、郑州、南宁等20余城市的首套房贷利率都是在LPR基础上加超100个基点；上海因此前房贷利率普遍有折扣，是唯一一个向下减点的一线城市，要求利率不低于相应期限LPR减20基点。

', '

\u3000\u3000不过，各地区的加减点数也随着LPR报价动态调整。新京报记者4月下旬咨询上海地区银行时，工行一位信贷人士称，首套房商贷利率为4.65%（3月时是在当月LPR报价基础上减10个基点），二套房利率最低是在LPR基础上加60个基点。易居研究院智库中心研究总监严跃进表示，上海最低利率和LPR利率已经持平了，这个时候不能再推行负基点的政策了，最低也只能按LPR基础利率走了。

', '

\u3000\u3000未来是否仍有降息空间？东方金诚首席宏观分析师王青认为，5月出于政策节奏等因素考虑，MLF及LPR降息过程暂停，但未来一段时间货币政策边际宽松还有空间，降息过程将会持续推进。温彬预计，6月开始降准和降息都还是有空间和必要的。他分析称，政治局会议明确将研究推出包括提高财政赤字率、发行特别国债、增加地方政府专项债券规模等，这些都需要全面降准，使银行获得长期低成本资金来配置，有助于降低无风险收益率的利率，引导债券市场直接融资成本的下降。

', '

\u3000\u3000新京报记者 程维妙
