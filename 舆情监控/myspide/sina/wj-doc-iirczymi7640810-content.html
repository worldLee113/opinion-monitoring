\u3000\u3000原标题：今年防汛形势不容乐观 成都将联合德阳资阳开展防汛大演练

', '

\u3000\u30004月21日，成都商报-红星新闻记者从成都市水务局获悉，随着汛期临近，成都已做好防汛“战斗”准备工作。从预判形势看，今年汛期，成都市强对流天气较常年偏多、偏重，极端性更强，防汛减灾形势不容乐观。

', '

\u3000\u3000据介绍，成都对前期发现的度汛隐患，将指定专人负责，抓紧完成整改。

', '

\u3000\u3000在江河防洪方面，将抓紧完成超标洪水防御预案编制，抓细抓实转移安置措施；加强江河顶冲段、流经城镇段、常年险情段重点监控，落实应急抢险措施；加快推进水毁工程建设，全力恢复江河行洪能力。

', '

\u3000\u3000在水库安全度汛方面，成都将加强李家岩水库、金马河二号闸、盘龙寺电站等重大在建水利工程和病险水库的监测监控和日常巡查管理，确保重点水利工程度汛安全。

', '

\u3000\u3000在山洪灾害防御方面，成都将结合区划调整，进一步完善山洪灾害防治责任体系；强化山洪灾害隐患点位排查评估和动态管理，明确预防预警程序和信号，完善预警信息传递最后一公里机制；持续开展山洪灾害防御宣传，加强应急转移演练，切实提升群众的应急避险和自救互救能力。

', '

\u3000\u3000在内涝防治方面，成都将加强城区危旧房、低洼棚户区、下穿隧道、地铁、地下车库等重要部位的预警防范，积极落实防汛排涝供电、治安、交通等保障措施，加强重点积水区、重要防涝区域的防汛物资储备和抢险力量准备。

', '

\u3000\u3000同时，成都还将加强冲锋舟、橡皮艇、抽水泵等应急抢险设施维护保养和检修，持续开展专项和综合防汛应急演练，对山洪灾害危险区、水库等重点部位应急演练必须要做到全域覆盖。

', '

\u3000\u3000今年5月，成都还将联合德阳、资阳开展防汛大演练，加强上下游防汛联动调度，确保安全度汛。
