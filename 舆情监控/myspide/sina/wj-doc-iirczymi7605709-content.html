\u3000\u3000原标题：湖北今年一季度GDP比上年同期下降39.2%

', '

\u3000\u3000新京报快讯\xa0据湖北省统计局微信公众号消息，面对突如其来的新冠肺炎疫情，全省上下切实把思想和行动统一到习近平总书记重要指示精神上来，认真贯彻落实习近平总书记考察湖北重要讲话精神，统筹做好疫情防控和经济社会发展各项工作，有力有序推进复工复产，努力把疫情造成的损失降到最低限度。

', '

\u3000\u3000根据地区生产总值统一核算结果，2020年一季度全省地区生产总值为6379.35亿元，按可比价格计算，比上年同期下降39.2%。其中，第一产业增加值为540.68亿元，下降25.3%；第二产业增加值为2146.96亿元，下降48.2%；第三产业增加值为3691.71亿元，下降33.3%。

', '

\u3000\u3000一、农业生产下降

', '

\u3000\u3000一季度，全省农林牧渔业增加值568.91亿元，下降24.8%。当季蔬菜播种面积288.83千公顷，下降2.6%，总产量823.16万吨，下降9.8%。生猪出栏544.97万头，下降52.6%。牛出栏26.13万头，下降13.9%。羊出栏86.38万只，下降53.2%。家禽出笼13244.43万只，下降23.8%。禽蛋产量53.55万吨，增长12.0%。渔业产量111.06万吨，下降18.1%。

', '

\u3000\u3000二、工业经济降幅收窄

', '

\u3000\u3000一季度，全省规上工业增加值同比下降45.8%，降幅比1-2月收窄0.4个百分点。其中，3月当月下降46.9%，降幅比2月收窄32.9个百分点。全省41个行业大类中，除烟草制品业、其他采矿业、石油和天然气开采业3个行业增加值保持正增长外，38个行业同比下降，其中27个行业降幅比1-2月收窄。基础行业中的电力、钢铁、有色等三大行业增加值分别下降18.2%、27.8%和39.4%，降幅分别小于规上工业27.6个、18个和6.4个百分点。与防疫需求相关的医药、纺织行业增加值分别下降42.1%和48.2%，分别比1-2月收窄3.5个和9.6个百分点；与生产生活保障相关的化工、农副食品加工行业增加值分别下降45.1%和52.3%，分别比1-2月收窄4.7个和5.3个百分点。装备制造行业中的汽车、电气机械、通用设备行业增加值分别下降56%、62.4%和59.9%，降幅分别大于规上工业10.2个、16.6个和14.1个百分点。

', '

\u3000\u3000三、投资活动放缓

', '

\u3000\u3000一季度，全省固定资产投资下降82.8%。基础设施投资下降84.4%；民间投资下降82.6%；工业投资下降85.0%，其中制造业投资下降85.0%，工业技改投资下降87.7%。全省房地产开发投资完成257.22亿元，下降73.7%，降幅小于全省平均水平9.1个百分点。全省商品房实现销售面积317.29万平方米，下降80.3%，实现销售额210.87亿元，下降85.0%。

', '

\u3000\u3000四、消费市场缓慢恢复

', '

\u3000\u3000一季度，全省共实现社会消费品零售总额2939.43亿元，同比下降44.9%。限上零售额下降49.1%，其中3月当月下降55.7%，降幅环比收窄25.8个百分点。分行业看，一季度全省限上批发业销售额、零售业销售额、住宿业营业额、餐饮业营业额分别下降37.7%、45.2%、48.1%、57.6%。从商品看，中西药品类限上商品零售额下降14.6%，粮油食品类、日用品类限上商品零售额分别下降19.4%、31.8%，汽车类、服装鞋帽针纺织品类、家具类、金银珠宝类限上商品零售额分别下降59.8%、60.6%、65.0%和65.1%。

', '

\u3000\u3000五、货物进出口减缓

', '

\u3000\u3000一季度，全省完成人民币计价进出口总额627.5亿元，下降20.9%，同比回落37.1个百分点。其中，出口总额318.3亿元，下降38.1%；进口总额309.2亿元，增长11.1%。

', '

\u3000\u3000六、财政收支下降

', '

\u3000\u3000一季度，地方一般公共预算收入完成542.49亿元，下降47.6%。其中税收收入434.04亿元，下降45.9%。地方一般公共预算支出1555.06亿元，下降14.1%。

', '

\u3000\u3000七、金融市场运行平稳

', '

\u3000\u3000截止3月末，全省金融机构各项存款余额为62629.98亿元，增长6.8%，比年初增加2882.28亿元。各项贷款余额为53070.88亿元，增长12.3%，比年初增加2406.92亿元。

', '

\u3000\u3000八、CPI继续上涨，PPI略有回落

', '

\u3000\u3000一季度，全省居民消费价格上涨6.1%，涨幅比1-2月提高0.1个百分点。全省工业生产者出厂价格与去年同期持平，环比回落0.1个百分点。全省工业生产者购进价格上涨0.5%，涨幅环比回落0.5个百分点。
