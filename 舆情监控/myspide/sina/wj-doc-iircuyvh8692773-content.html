\u3000\u3000原标题：浙江仙居发现珍稀濒危野生动物中华斑羚

', '

\u3000\u3000中新网台州4月19日电（记者 范宇斌 通讯员 周国平）灰色针毛、黑色背纹、圆圆的眼睛、高超的攀岩能力……19日记者获悉，浙江省森林资源监测中心专家在台州仙居括苍山省级自然保护区进行红外相机数据收集，意外发现红外线相机拍摄到珍稀濒危野生动物中华斑羚的高清视频。

', '

\u3000\u3000仙居括苍山省级自然保护区保护站站长翁钱塘介绍，拍摄地位于海拔800多米的仙居括苍山自然保护区，保护区内安装了140个红外相机，拍摄时间是2019年12月19日。因为新冠肺炎疫情，科考人员一直没有来，4月16日取卡后发现野生中华斑羚。

', '

\u3000\u3000据悉，此次多个相机位点记录的大量视频和照片充分证实了仙居是中华斑羚华东种群的重要集中分布区。

', '

\u3000\u3000中华斑羚为国家二级重点保护动物，IUCN红色名录和《中国生物多样性红色名录》易危物种。由于全国范围内中华斑羚已处于濒危状态，分布区片段化过程不断加剧，其中最东端的浙江种群与中部种群割裂严重，中间已不连续，浙江分布区呈孤岛状。因此，仙居中华斑羚的发现对该物种多样性的保护具有十分重要的意义。

', '

\u3000\u3000据了解，中华斑羚在国内曾分布较广。近几十年来，由于人类活动干扰以及猎捕导致种群数量急剧下降，华东、华南已很难发现它的踪迹。目前，浙江境内也只有仙居括苍山省级自然保护区和丽水莲都峰源省级自然保护区有红外相机的野外记录，并且记录的点位稀少。

', '

\u3000\u3000仙居括苍山自然保护区坐落在仙居国家公园内，面积45000余亩。2014年3月，仙居国家公园先后与多家高等院校和科研单位的对自然保护区生物多样性的保护与研究。

', '

\u3000\u30002015年5月仙居县发布了全国首个县级生物多样性保护行动计划，设置永久相机用热感红外无人相机定位技术开展生物多样性调查与监测，共发现大型鸡类白颈长尾雉、白鹇、兽类斑羚等国家一级、二级保护动物7种中大型兽类和4种保护鸟类和珍稀植物。

', '

\u3000\u3000仙居国家公园管委会副主任朱瀚波表示：“多年来，仙居国家公园注重生态保护与科研相结合，让自然保护区既成为生态绿肺，又成为生物多样性研究永不移动的教科书，所以近年来陆续发现了不少珍稀物种。”（完）
