\u3000\u3000原标题：T179次列车在京广铁路湖南永兴段脱轨，有人员受伤

', '

\u3000\u3000新京报讯（记者 康佳 张彤 王洪春）3月30日中午，由济南发往广州的T179次列车，途径湖南省永兴县境内时，发生脱轨事件，并有车厢起火。中国铁路广州局集团有限公司官方微博发布消息称，事因线路塌方，目前，受伤的铁路员工和旅客已送医得到治疗。

', '

\u3000\u3000现场视频显示，一辆列车停在铁轨上，其中两节脱离轨道，稍远处有车厢起火，现场有明火和浓烟冒出。新京报记者查询到，从济南开往广州的客运火车车次中，T179次列车于12时11分时正好途经郴州。

', '

\u3000\u3000新京报记者从郴州市委宣传部负责人处了解到，T179次列车在永兴县内发生脱轨，伤亡情况仍未知。

', '

\u3000\u3000今日13时许，新京报记者从湖南郴州市应急管理局工作人员证实，从济南开往广州的T179次列车在郴州永兴县高亭司镇境内发生侧翻，有车厢脱轨，机头着火。该工作人员称，相关部门已派人前去处理，暂不清楚是否有人员伤亡。

', '

\u3000\u3000郴州市120急救中心工作人员告诉新京报记者，已派出10辆救护车前往救援，伤者尚未送返医院。

', '

\u3000\u3000中国铁路广州局集团有限公司官方微博发布消息称，京广铁路马田墟至栖凤渡区段因塌方致T179次脱线：今日11时40分，受连日降雨影响，京广线马田墟至栖凤渡站下行区间发生线路塌方，T179次（济南至广州）列车运行至该区段时，火车司机发现后立即采取紧急制动措施，列车撞上塌方体，导致机后第一节发电车起火，第二至六节车厢脱线倾覆。铁路部门第一时间启动应急预案，抢险救援工作已经展开。经现场检查确认，事故未造成人员死亡，受伤的铁路员工和旅客已送医得到治疗，具体原因正在调查中。
