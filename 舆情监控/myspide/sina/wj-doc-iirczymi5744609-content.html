\u3000\u3000原标题：民政部：以四项部署保障特殊困难群体的日常生活

', '

\u3000\u3000中新社北京4月11日电 （记者 李亚南）中国民政部基层政权建设和社区治理司司长陈越良11日在北京表示，进入常态化疫情防控阶段，民政部将以四项工作部署保障特殊困难群体的日常生活。

', '

\u3000\u3000中国国务院联防联控机制当日召开新闻发布会，介绍优化社区防控筑牢安全屏障工作情况。

', '

\u3000\u3000陈越良在发布会上表示，与其他社区居民相比，一老一小、残疾人等特殊困难群体居家生活面临更多的困难，是最需要关心关爱的群体。新冠疫情发生以来，民政部门对这些特殊困难群体的生活照料作出部署，有效保障了社区特殊困难群体的日常生活。

', '

\u3000\u3000他介绍称，新形势新任务下，民政部将进一步动员社会各方面力量，共同做好社区特殊困难群体关心关爱工作。

', '

\u3000\u3000一是恢复秩序。推动各地精准实施分区分级差异化防控策略，逐步恢复社区正常生活秩序。特别是一些低风险地区要有序逐步开放必须的生活服务设施，为一老一小、残疾人等特殊困难群体提供服务，为复工复产创造条件。

', '

\u3000\u3000二是就地就业。在需要的地区采取政府购买服务方式，开发疫情防控非全日制临时社区工作岗位，既帮助经济困难的社区居民临时就业，又增加社区防控和服务工作力量。

', '

\u3000\u3000三是购买服务。制定完善相关工作指引，在社区发挥社会工作专业优势，支持社工、义工、志愿者开展情绪支持、心理疏导等服务，引导各方面力量共同做好特殊困难群体的帮扶。

', '

\u3000\u3000四是工作创新。在总结前段社区疫情防控工作有效经验的基础上，加强制度设计，推动城乡社区为特殊困难群体提供更为精细化、精准化、智能化的生活服务，包括家政服务。（完）
