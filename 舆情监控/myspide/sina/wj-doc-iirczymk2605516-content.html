\u3000\u3000原标题：中物联：超半数被调查物流企业一季度处于亏损状态

', '

\u3000\u3000新京报讯（记者 程子姣）5月20日，中国物流与采购联合会（下称中物联）正式发布《2020年物流与供应链企业复工达产营商环境调查报告》，报告显示，79.8%的企业在3月上旬之前全面复工。一季度行业企业亏损面较大，51.7%的企业处于亏损状态，超过被调查企业的半数。

', '

\u3000\u3000业务量方面，一季度物流企业业务量大幅下降，68.1%的企业业务量比去年同期出现减少，其中79%的小微企业业务量呈减少状态，45.2%的民营企业业务减少在20%以上。

', '

\u3000\u3000在税务环境方面，报告显示，企业税负水平有所减少，一季度物流企业共缴纳各项税金及附加税占主营业务收入的比重平均为10.5%。税收减免政策惠及企业，共有59.6%的企业享受或部分享受增值税、企业所得税和土地使用税税收优惠政策，部分或全部享受各项政策的企业均超过30%。有38%的企业享受或部分享受增值税减免政策。

', '

\u3000\u3000但也有企业反映（税收减免）政策覆盖范围较窄，仅限于运输疫情防控重点保障物资取得的收入和小规模纳税人。

', '

\u3000\u3000在通行环境方面，报告称，道路通行环境明显改善。随着2020年1月1日起，全面取消高速公路省界收费站，有效缓解了高速公路车辆通行问题。通行费免收政策助力复工复产，超过89%的调查企业认为阶段性免收收费公路车辆通行费对企业复工复产有积极影响。

', '

\u3000\u3000报告还指出，目前物流及供应链企业面临的主要困难包括，需求不足、税费偏高和资金压力。企业经营不景气仍将持续，行业亏损面预计较大。企业投资方向正逐步转变，数字物流新型基础设施成为投资方向。各企业有信心完成全年目标，行业营商环境预期总体向好。

', '
