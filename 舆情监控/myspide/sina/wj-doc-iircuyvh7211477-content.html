\u3000\u3000原标题：杭州三辆公交车印792名援鄂医疗队医护人员名字

', '

\u3000\u3000来源：北京青年报

', '

\u3000\u3000最近几天，许多杭州市民都在路上看到了三辆涂装特殊的公交车，这三辆公交车上密密麻麻地印着792个名字，而名字被印上公交车的这些人，都是杭州援鄂医疗队的队员，杭州公交集团在选取公交车的时候，特意选择了三条途经医院较多的线路，为的就是能够让更多的医护人员看到这份特殊的“礼物”。

', '

\u3000\u3000据了解，4月3日，最后一批杭州援鄂医疗队返回，4月5日，杭州公交集团便推出了这三辆印满援鄂医疗队队员名字的公益巴士。

', '

\u3000\u3000“这三辆车分别来自156路、49路和183路，这三条线路平时经过的医院会比较多，4月5日当天这三台车就上路行驶了。”杭州公交集团工作人员介绍说，这三辆公交车的车身设计以深蓝色、正红色为主基调，意喻援鄂医疗队队员的名字如同星星在夜空中闪烁一样，也象征医护人员艰苦卓绝的付出，撕开疫情笼罩的大幕，带来的是桃红柳绿、生生不息的春天的希望。

', '

\u3000\u3000北青报记者了解，在公交车身上被印上名字的援鄂医护人员来自浙江省人民医院、杭州市第一人民医院、浙江大学医学院附属第一医院、浙江大学医学院附属第二医院、浙江大学医学院附属邵逸夫医院、浙江省中医院等医院，共有792名。

', '

\u3000\u3000杭州183路公交车司机郭师傅所驾驶的公交车便是其中印满援鄂医疗队队员名字的公交车之一。“一路上也经常能看到有市民拿出手机在拍摄这辆车，能够开上这样的大巴车感觉还是很光荣的，让这些医护工作者的名字在我们这座城市传播，自己也很骄傲。”
