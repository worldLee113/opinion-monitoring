\u3000\u3000原标题：两会关注|政协委员莫天全：跨部门监管住房市场有序竞争

', '

\u3000\u3000新京报快讯（记者 张晓兰）2020年全国两会即将在北京召开。全国政协委员、房天下董事长莫天全在建议中提出，一轮又一轮房地产宏观调控政策的出台，使得中国房地产市场相对稳定，但是仍然存在着住房市场过度集中的风险，需要建立跨部门的监管机构来确保全国住房市场的健康发展。

', '

\u3000\u3000市场集中度趋高，房价扭曲风险提升

', '

\u3000\u3000目前我国房地产市场开发、销售过程中，房企、中介机构等的集中度越来越高，有形成寡占市场结构的趋势。

', '

\u3000\u3000莫天全指出，当行业良性竞争局面被打破，就意味着垄断机构有收费和对房价的话语权和定价权，对控制房价非常不利，各种费用会不断升高，最终受害者是处于弱势地位的消费者（购房者），不利于贯彻落实“房住不炒”的指导方针。

', '

\u3000\u3000莫天全表示，目前，银行业、保险业、证券业等都已经建立了国家体系的监管机构。银保监会、证监会等对银行业、保险业和股票等证券业等市场的审慎有效集中统一的监管，对我国相关行业和市场的健康良性发展，起到了非常大的作用，值得住房市场借鉴。

', '

\u3000\u3000建立跨部门国家住房市场监管机构

', '

\u3000\u3000我国的住房市场，关系到民生中最重要的住的问题。而我国的住房市场规模，大大超过银行业、保险业、证券业的市场规模。

', '

\u3000\u3000莫天全建议，建立跨部门的国家住房市场监管机构。组建国家住房市场监管机构，依照相关法律、法规统一监督、管理、调控全国房地产市场，预防部分城市逐渐形成的中介机构高市场占有率的垄断格局，加强对寡头企业监管，保障行业回归到健康有序的竞争中来。

', '

\u3000\u3000同时，莫天全指出，建立全面的国家住房市场监管制度，让国家住房市场监管机构有规可依、有法可依，让所有住房市场的参与者，都能够不仅受到监管，而且得到支持和保护，保障住房市场的充分良性竞争，杜绝住房市场垄断势力的出现，确保“房住不炒”，房价有序，住房交易费用合理。
