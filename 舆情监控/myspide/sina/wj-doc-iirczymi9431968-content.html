\u3000\u3000不少人可能都注意到，五一小长假第一天，国务院应对新型冠状病毒肺炎疫情联防联控机制综合组发布了一份通报。

', '

\u3000\u3000通报的名称是《关于黑龙江省新冠肺炎聚集性疫情有关情况的通报》，这份通报，是发给全国的。

', '

\u3000\u3000而就在5月1日，黑龙江省委书记、省应对新冠肺炎疫情工作领导小组组长张庆伟到了哈尔滨市香坊区，督导检查疫情防控工作。

', '

\u3000\u3000“存在厌战情绪和麻痹松懈思想”

', '

\u3000\u3000通报中提到，2020年4月以来，黑龙江省哈尔滨市、牡丹江市发生境外新冠肺炎输入病例关联的本土聚集性疫情，特别是涉及多家医院先后发生院内感染，持续时间长，感染病例多，社会影响恶劣，给巩固来之不易的疫情防控成果带来新的压力。

', '

\u3000\u3000一组数据也可以佐证。

', '

\u3000\u3000今天，国家卫健委新闻发言人米锋说，4月份，全国共报告新增本土确诊病例131例，其中超过60%来自黑龙江本土聚集性疫情。

', '

\u3000\u3000上述通报中还提到了黑龙江省暴露的四个问题：

', '

\u3000\u3000对疫情防控形势认识不足

', '

\u3000\u3000院前防控措施存在薄弱环节

', '

\u3000\u3000核酸检测工作未能按要求及时开展

', '

\u3000\u3000院内感染防控措施落实不力

', '

\u3000\u3000通报提到，哈尔滨市、牡丹江市作为口岸城市，未能充分认识“外防输入、内防反弹”的严峻形势，对境外输入疫情风险认识不足，存在厌战情绪和麻痹松懈思想。

', '

\u3000\u3000就在4月30日，黑龙江方面曾召开会议，审议了《深入贯彻落实习近平总书记重要讲话精神开展查漏洞补短板大排查工作方案》。

', '

\u3000\u3000方案明确，要在全省全面开展疫情防控漏洞短板大排查。

', '

\u3000\u3000这次会议还提到，要坚决克服麻痹厌战松劲侥幸心理，时刻绷紧疫情防控这根弦，决不能贻误战机、一错再错，决不能让疫情防控工作功亏一篑。

', '

', '

\u3000\u3000黑龙江省森工总医院（省红十字医院）

\u3000\u3000黑龙江中医药大学附属第一医院

', '

\u3000\u3000勃莱梅酒店

', '

\u3000\u3000比如，相关医院错误地认为，经海关和社区防控能够完全筛出新冠病毒感染者，发生院内聚集性疫情的可能性极小，存在侥幸心理。

\u3000\u3000医务人员防护意识淡薄、敏感性不强、培训不到位等

\u3000\u3000张庆伟这次督导检查的内容，就是“预检分诊、就诊管理等工作”。

\u3000\u3000据当地媒体披露，在提到当前黑龙江疫情防控形势时，张庆伟说，“依然严峻、压力很大”。

', '

\u3000\u3000张庆伟要求，各级领导干部要迅速行动起来，带头深入一线、靠前作战，认真开展大排查，及时发现漏洞，加大整改力度，加快补齐短板。

', '

\u3000\u3000医院在疫情防控中责任重大，不能有半点含糊，要坚持住、不松劲，既要打好阻击战，也要做好常态化疫情防控，各岗位责任人都要死看死守、加强巡查检查，确保各项规章制度落到实处。

', '

\u3000\u3000张庆伟提到，严格落实预约挂号、预检分诊、转诊转院等制度，严格执行诊疗流程，加强住院患者及陪护人员管控，规范病房管理，加快提高核酸检测能力，做好重点部位医学消杀，坚决防止医院再出现交叉感染。
