\u3000\u3000（抗击新冠肺炎）河南远程连线利比里亚华侨华人 分享疫情防控经验

', '

\u3000\u3000中新网郑州4月17日电 （记者 韩章云）4月16日傍晚，一场防控新冠肺炎疫情专家视频会在郑州举行。视频会议的一方是河南省经验丰富的高水平卫生防疫专家，另一方则是远在非洲西部的利比里亚中资企业、华侨华人代表。

', '

\u3000\u3000根据外交部工作安排和中国驻利比里亚使馆建议，河南省与中国驻利比里亚使馆共同举办此次视频会议。

', '

\u3000\u3000会议于北京时间4月16日17：30（利比里亚当地时间4月16日9：30）举行。河南分会场设在河南省互联网医疗监管办公室，利比里亚分会场设在中国驻利比里亚使馆、利比里亚中资企业驻地和当地华侨华人家中。河南省委外办、河南省卫生健康委、郑州大学第一附属医院相关负责同志、医疗专家等在河南分会场参加会议。

', '

\u3000\u3000出席当日视频会议的河南省委外办主任付静表示，河南老乡们十分关注利比里亚的疫情，也十分牵挂常年驻利比里亚同胞们的健康与安全。“我们希望视频会能够让各位同事、各位同胞有所收获；各位同胞增强防范能力、保护好身体健康是我们最大的心愿；我们也愿意为同胞们提供力所能及的协助。”

', '

\u3000\u3000记者了解到，河南省选派郑州大学第一附属医院两名一线经验丰富的高水平卫生防疫专家参加现场连线。专家们向在利中资企业、华侨华人代表介绍疫情防控知识，并针对他们在新冠肺炎防控具体实践中遇到的困难进行了解答。

', '

\u3000\u3000利比里亚中资企业援外机构和利比里亚华侨华人代表对来自祖国的关心和支持非常感谢。他们表示，来自老家河南的专家展现出一流的医疗水平，解答了他们很多困惑，为他们下一步疫情防控工作提供了有力支持。

', '

\u3000\u3000据介绍，河南和利比里亚虽然在距离上十分遥远，但双方一直保持着非常良好的交往合作。2017年以来，利比里亚多位政要以及驻华大使等先后到访河南，考察项目、推动合作、促进交流。

', '

\u3000\u3000目前，中国电建集团河南工程有限公司、河南国际合作集团、许昌瑞美真发公司等多家河南企业在利比里亚开展了富有成效的项目投资合作。（完）
