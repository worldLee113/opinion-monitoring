\u3000\u3000原标题：北京6家公园游船明起陆续开航，实名登记、限流乘坐

', '

\u3000\u3000颐和园、玉渊潭、北海公园、中山公园、陶然亭公园与紫竹院公园6家有游船的市属公园4月18日起将正式开航。记者今天从北京市公园管理中心获悉，6家公园正配合相关部门开展验船、防疫等各项准备工作，在天气情况允许的条件下，明天将正式开航，为市民提供更多休闲游憩新选择。

', '

\u3000\u3000今天上午，记者在颐和园玉澜堂码头看到，市交通委员会海淀运输管理分局的工作人员正在对码头上的救生、消防、系缆设备和防滑设施，以及230余条各类游船的救生设备、电瓶等进行逐一检查。“每年开航前，交通部门都要对游船进行年度审核，这是保障游客安全的基础，但是今年情况特殊，在保障游船安全的基础上，我们还要做到防疫安全。”颐和园管理处相关负责人说。

', '

\u3000\u3000记者看到，在玉澜堂码头的服务窗口前，工作人员正在地上贴一米线，“我们会有专人疏导游客，按照一米线排队，让游客保持安全距离。”该负责人告诉记者，游客在买票环节需要提供健康码等信息，并实名登记个人信息，才能购票。如果遇到节假日大客流，公园会加大疏导力量，用发号等方式，鼓励游客分散等候。

', '

\u3000\u3000购票上船后，防疫工作也随处可见。该负责人带着记者来到一艘小型游船旁边。这艘脚踏船限乘6人，“根据防疫要求，6人船只能坐4个人，4人座的小船则不超过3人，游客需要全程佩戴口罩，不能拼船。我们会在座位上贴出标识，让游客间隔乘坐。”

', '

\u3000\u3000对于乘坐数量接近百人的画舫和摆渡船来说，限乘的要求更加严格，船上不仅会配备安全员，还会按照乘客量不超过载客量的50%限流乘坐，游客分散、错位就座。

', '

\u3000\u3000目前颐和园共有4个小船码头和7条大船航线，明起将先开通玉澜堂小船码头和铜牛到石丈亭的大船航线，五一前后将陆续开通其他码头和航线。

', '

\u3000\u3000据悉，为了确保游船的接待安全，开航之后每天还会进行例行检查。每天早上，游船管理人员都会从气象台获取当日的气象报告，风力大于4级的时段将停止出船。

', '

\u3000\u3000除了颐和园，北海公园、中山公园、陶然亭公园、紫竹院公园和玉渊潭公园的游船也正在开展验船、防疫等各项准备工作，市公园管理中心服务处处长缪祥流介绍，若明天天气条件合适，从本周末起，市属6家公园将陆续开放游船码头，迎来2020年首航，画舫、手划船、脚踏船、电动船等4类、1700条大小船只将陆续投入运营。

', '

\u3000\u3000为确保市民游客安全乘船游览，各公园所有运营船只每日集中消毒2至3次以上。每日开航前先完成1次集中消毒作业，运营中根据游客量情况再进行1到2次的消毒清洁，以确保运营船只的清洁卫生、健康安全。

', '

\u3000\u3000市公园管理中心提醒市民，现阶段6家市属公园的游船正在逐步恢复运营，目前数量还相对有限，请游客避开乘船的高峰时段，如遇排队请配合现场工作人员的管理工作。目前公园湖上温度仍然较低，建议游客租用船只时间以1小时为宜，老人、儿童上下船需要注意安全。

', '
