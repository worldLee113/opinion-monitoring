\u3000\u3000原标题：4.94万台，3月挖掘机销量创历史纪录！透露什么信号？

', '

\u3000\u3000来源：每日经济新闻

', '

\u3000\u300049408台，一个崭新的纪录诞生！

', '

\u3000\u3000近日，中国工程机械工业协会发布的统计数据显示，2020年3月，全国共计销售各类挖掘机械产品49408台。去年3月，全国挖掘机销量曾创下当时的销量最高纪录44278台，最新数据在原纪录基础上增长了11.6%。

', '

\u3000\u3000挖掘机被视作工程机械行业的风向标。此前在今年1月和2月，国内市场挖掘机销量分别为7758台和6909台，同比分别下降23.5%和60%。

', '

\u3000\u3000而3月份挖掘机销量回升，既符合天气转暖后开工增加的客观规律，一定程度上也反映出疫情影响趋于消散，需求释放加速推进。

', '

\u3000\u3000当前哪一类型挖掘机销售增长最为突出？哪些需求有望进一步带动挖掘机销售？机构对未来挖掘机市场情况是否持续看好？

', '

\u3000\u3000小型挖掘机销量增幅最大

', '

\u3000\u3000随着复工复产工作进一步推进，3月挖掘机需求逐渐释放，销量实现两位数增长。

', '

\u3000\u3000根据中国工程机械工业协会的统计，在3月份销售的49408台各类挖掘机械产品中，国内市场销量占到46610台，同比增长11.2%；出口销量2798台，同比增长17.7%。

', '

\u3000\u3000国元证券研报表示，分结构看，大/中/小挖掘机销量同比增速分别为3.66%、-0.03%、18.49%，其中小型挖掘机销售增幅尤为突出。这类产品主要应用在农村建设、房屋改建等投资较小的现场，销量弹性高，对复工反应最为迅速。

', '

\u3000\u3000中信建投分析认为，3月中旬以来，挖掘机销量快速上升，部分机型已经开始出现缺货现象，在重点项目复工率基本到位的情况下，今年4月可能为销量高点，预计4月挖掘机销量同比增长55%，2020年总销量同比增长8.87%。

', '

\u3000\u3000挖掘机销售作为产业链中游指标，能够反映下游基建和房地产投资需求的强弱。

', '

\u3000\u3000数据显示，今年1~2月，全国固定资产投资（不含农户）33323亿元，同比下降24.5%，基础设施投资（不含电力、热力、燃气及水生产和供应业）同比下降30.3%。

', '

\u3000\u3000与之相应，1~2月挖掘机销量也出现明显下降。据中国工程机械工业协会统计，今年2月，纳入统计的24家挖掘机制造企业共销售各类挖掘机9280台，同比下降50.5%。其中，国内销量6909台，同比下降60%。

', '

\u3000\u30002020年1月纳入统计的25家挖掘机制造企业，共计销售各类产品9942台，同比下降15.4%，其中国内市场销量7758台，同比下降23.5%。

', '

\u3000\u3000葛昕表示，二季度工程机械市场将快速升温，主要是因为疫情影响造成很多工程进度严重滞后，市场存在赶工需求，所以挖掘机销量有可能超出市场预期，目前预计二季度挖掘机市场销量有望同比增长40%，其中4月份销量更有望增长50%。

', '

\u3000\u3000重大项目复工有望提振后续市场

', '

\u3000\u30003月以来，在国内疫情防控形势明显好转的情况下，政策目标调整到统筹疫情防控与经济社会发展，重在畅通供给循环。近期先行指标如PMI回暖、高频指标如高炉开工率和发电耗煤量上升等，都反映复工复产在进一步加快。

', '

\u3000\u3000罗志恒表示，当前专项债加快发行，地方政府加快推进项目建设保增长和就业。从挖掘机类型看，小型挖掘机销售增速最为显著，达到18.4%，这与解决部分农民工就业、加快农村基础设施建设有关。

', '

\u3000\u3000未来挖掘机销售的趋势如何？罗志恒认为，当前复工复产继续加快，加上基建稳增长等政策，为了将疫情对经济的影响降到最低，4月挖掘机销量仍可能继续上升。

', '

\u3000\u3000记者注意到，科尔尼管理咨询公司日前发布的报告显示，新基建带来的需求或将重塑工程机械行业产品布局。对于城市中的小型工程建设，如新能源充电桩、5G基站建设等，对于小微型挖掘机的需求有望产生提振作用。
