', '

Monica是美国得克萨斯州一家康复中心的职业治疗师，17年间帮助上万名病人恢复健康。她在新冠肺炎疫情刚开始爆发时就非常重视，时刻关注疫情，并有意识地佩戴口罩工作。

', '

3月初，她所在的医院有医务人员向院方要求提供口罩而被解雇。Monica也因佩戴口罩工作而被院方叫去问话，并被要求摘下口罩。随后，她决定离职。

', '

4月14日，在接受潇湘晨报记者采访时，Monica说，“医院根本不顾我们的安危。我有一个17岁的孩子，她很依赖我，我对她的照顾比对病人的照顾更重要。”

', '

这个孩子，是Monica从中国收养的女儿。作为独自将女儿抚养长大的单身妈妈，她选择将孩子放在首位。

', '

“我很难过我没能提供帮助，但我没有遗憾。我选择收养我的女儿，就要对她负责，让她在没有母亲的情况下离开，这是不能接受的，尤其是我本能够防止这种情况的发生。”

', '

据美国约翰斯・霍普金斯大学公布的数据显示，截至14日，美国已有超过60万人感染新冠肺炎，死亡25575例，共检测3081620例。

', '

对话当事人

', '

【1】戴口罩被批评和嘲笑

', '

潇湘晨报：你所居住的城市现在疫情如何？

', '

Monica：我居住在得克萨斯州圣安东尼奥（San Antonio）。得克萨斯州目前有12500个病例，250多人死亡，每天约有700个新病例。圣安东尼奥目前有665个确诊病例。

', '

潇湘晨报：当地的第一个确诊病例是？

', '

Monica：圣安东尼奥的第一个确诊病例在2月10日左右。一架美国的大型飞机上有人被检测出携带冠状病毒。这也是美国的第十五例。

', '

潇湘晨报：你的工作是什么？

', '

Monica：我是一家康复中心的职业治疗师。我从事这份工作17年了。

', '

潇湘晨报：什么是职业治疗师？

', '

Monica：在美国康复医院，医生维持病人的医疗稳定，并指导他们康复。护士提供药品，听从医生的命令。物理治疗师和职业治疗师提供康复治疗，使病人更强壮，更有能力返回日常生活，最好是回归到他们本来的生活。我们主要处理中风、脑外伤及术后病人如心脏手术、髋关节手术、膝关节手术等。

', '

我们拥有硕士或博士学位。在每一个医疗机构，从急症医院、康复医院、门诊诊所、疗养机构到养老院都可以看到我们的身影。

', '

潇湘晨报：为什么离职？

', '

Monica：我戴着口罩上班，被员工批评和嘲笑。当我换上N95后，我的主管私下把我带到她的办公室，质问我为什么要戴口罩。我问她，这有什么关系？她告诉我，她需要知道是不是因为冠状病毒，因为我们医院没有新冠患者。我说我不能告诉她，因为这样做违反了HIPAA（健康保险携带和责任法案）。然后她请传染病科主任到办公室来共同质问我，并要求我摘下口罩。我感到自己并不被尊重。

', '

在我被叫到办公室之前，我的一个同事因为医院不给提供口罩而离开，他们把他开除了。

', '

我有一个17岁的孩子，她很依赖我，我对她的照顾比对病人的照顾更重要。所以我没有回去工作了。

', '

潇湘晨报：你是什么时候被问话？

', '

Monica：在3月17日当天。

', '

潇湘晨报：你是怎么离职的？

', '

Monica：事实上，我不是全职工作。我们有一个APP，可以在上面选择自己的工作时间。 这件事发生后，我就把这个月剩下的时间和整个4月份的工作时间都取消了。

', '

我不知道在这之后，我还能不能找到工作。在我离开一周后，医院里大概有五个医护人员也辞职了。

', '

【2】每月可领一些救助金

', '

潇湘晨报：是否会感到遗憾？

', '

Monica：如果我没有女儿，我会毫不犹豫地照顾新冠患者，我爱我的工作。

', '

我是一个单身妈妈，只要是这么危险的病毒，我就得把女儿放在第一位。我们中的任何一个人都可能被感染，但在医院里更有可能。

', '

我很难过我没能提供帮助，但我没有遗憾。我选择收养我的女儿，就要对她负责，让她在没有母亲的情况下离开，这是不能接受的，尤其是我本能够防止这种情况的发生。

', '

潇湘晨报：你身边的人如何看待？

', '

Monica：我的同事们同意我的观点，他们大多数都已成为父母。夫妻二人都是医护人员的家庭中至少有一个已经辞职了。

', '

如果我是一个人，我不会害怕在我的岗位上为别人而死。但收养了我的女儿后，她对我来说就是最重要的，我不愿让她再次经历失去亲人的痛苦，我要首先考虑她。

', '

我的心很痛，我曾经和那些濒临死亡的人在一起，知道感染了这种病毒，家人不允许陪同，这些病人只能独自死去。我希望我能在那里牵着他们的手，陪伴他们。

', '

潇湘晨报：院方有联系你吗？

', '

Monica：他们没有因为我取消工作而联系我。我很确定他们知道我为什么要这么做。

', '

潇湘晨报：辞职后有压力吗？

', '

Monica：是的，失去这份工作的收入对我来说生活会变得困难。我没有全职工作是因为我有其他的投资收入，但还不足以维持我的生活。

', '

如果这种病毒在接下来的两个月内不消失，我从投资中赚到的钱也会丢失。所以我的时间有限，压力很大。

', '

潇湘晨报：政府有补贴吗？

', '

Monica：政府颁发了一项文件，因受冠状病毒影响而辞职的个人可获得失业援助。这项措施在这段时间里提供了帮助。

', '

潇湘晨报：你有领取到失业救助金吗？

', '

Monica：有，我每个月可以领取一些失业救助金。通常情况下，我们只有在被解雇的情况下才能申请失业。

', '

潇湘晨报：之后医院有没有做出什么改变？

', '

Monica：在他们把我叫去问话的三天后，医院发现有病人被检测出感染了冠状病毒，是被治疗他们的医生传染的。病人被隔离起来，整个医院才开始戴口罩，但是是外科口罩，而不是N95。

', '

潇湘晨报：疫情结束后你还能回到之前的医院继续工作吗？

', '

Monica：医院的事就不一定了，疫情结束后他们可能不想让我回来。谁知道新冠病毒什么时候会消失，现在一团糟。

', '

我也不确定我是否想回到医疗领域。如果能保障医护人员的安全，我愿意接受少一点的薪资。

', '

【3】有人因要戴口罩被解雇

', '

潇湘晨报：为什么医院不让你们戴口罩呢？

', '

Monica：我不确定答案。我们医院有传染病病人，当我们接触这些病人时，必须戴上口罩、手套和防护服进行防护。

', '

我可以推测的是，由于美国没有足够的个人防护设备来处理冠状病毒，他们试图限制口罩的使用，以便为冠状病毒患者提供。我相信我们有权利保护自己。但不幸的是，他们不愿意提供防护用品，也不希望我们从外面带来防护用品，因为可能会受到感染。

', '

潇湘晨报：别的医院也有这样的情况吗？

', '

Monica：对，我确定在很多地方都是这样，医护人员被告知不能戴口罩，也没有其他的防护用品。

', '

潇湘晨报：当时医院里的其他人有戴口罩吗?

', '

Monica：他们把我叫去问话的时候，楼里只有我和李医生戴着口罩。他们也找李医生谈过，李医生告诉他们可以开除她，但她不打算摘掉口罩。

', '

潇湘晨报：为什么当时你会戴口罩？

', '

Monica：在我们医院，如果我们在流感季节没有注射流感疫苗，就必须戴上外科口罩。我已经五年没有打过流感疫苗了，所以每年医院里的人都会看到我由于这个原因戴着口罩。

', '

去年12月，我的白细胞数量不对，那是免疫系统出现了问题。于是我问楼里的一位医生，是否应该打流感疫苗。她说需要，于是我就打了。

', '

但我并没有因此而停止戴口罩。11月和12月，没有人在意这件事，我继续戴着口罩，也没有人质疑我。当我开始戴N95口罩的时候，他们仍然没有质疑我。而我的一个同事因为想要戴口罩，遭受到医院的拒绝，并被解雇了。

', '

潇湘晨报：为什么从外科口罩换到N95？

', '

Monica：我不是全职工作，我很熟悉病毒传播的速度，以及如何被感染病毒。我的免疫系统有些问题，我很担心也很小心，所以从外科口罩换成了N95。

', '

潇湘晨报：你们医院现在的情况如何？

', '

Monica：我们医院主要负责康复治疗，受疫情影响，该市不再允许进行择期手术，甚至不允许与医生进行择期预约。所以不再有牙医、视力医生，也不再有可以推迟的手术。这类患者都是在手术后来到我们的康复医院。因此，我们的大楼空了一大半。

', '

【4】患者治疗费用由政府承担

', '

潇湘晨报：目前美国的医疗物资够吗？

', '

Monica：美国仍然存在医疗物资短缺问题。当前，政府希望每个人都戴口罩，但不要戴医用口罩。他们想让医用口罩去医院，所以他们希望我们要么自己制作一个，要么用布。

', '

潇湘晨报：为什么医疗物资仍短缺？

', '

Monica：受疫情影响，很多国家不得不关闭对外通道，很多工厂停止运作，不少运往美国的货物也都停止了。

', '

每个国家都在努力保护本国公民的生命安全，我们能够理解。我相信我们的国家仍然希望和中国保持良好的贸易关系，这并不仅仅是为了个人防护用品。

', '

潇湘晨报：医院里的医护人员呢？

', '

Monica：到目前为止，有足够的医护人员，因为政府取消了很多手术。但是，一旦我们的感染人数上升，就不确定了。

', '

潇湘晨报：政府有采取什么措施应对疫情吗?

', '

Monica：州政府已经停止了所有的选择性手术。取消手术的病人占到了50%。他们正在改建附近的一些养老院，供新冠肺炎患者使用。

', '

潇湘晨报：应对效果如何？

', '

Monica：自1918年以来，还没有病毒对这个世界造成如此严重的影响。每个国家都有自己的问题，每个国家都在尽最大努力处理正在发生的事情。如果没有身处其中，或许你会认为我们处理的不是那么糟糕。

', '

潇湘晨报：你所居住的城市有出现“囤抢物资”的现象吗？

', '

Monica：我相信有些人会这么做。但在我们城市，消毒产品和防护产品有限，如果大家都这么做的话，会造成很严重的隐患。

', '

潇湘晨报：据《纽约时报》报道，在美国富人和名人更容易优先检测，特朗普回答“这就是人生”你怎么看待？

', '

Monica：病毒是不会知道你银行账户里有多少存款的。现在所有生病的人都可以去做检查，任何人都可以得到病毒检测。

', '

我认为这是媒体提出的带有戏剧性的问题，所以特朗普也给出了一个带有戏剧性的答案。我相信别的国家富人也会更快得到检测。

', '

潇湘晨报：冠状病毒的治疗费用是个人支付还是政府承担？

', '

Monica：所有冠状病毒的治疗费用都由政府承担。他们之所以这样做，是因为有很多人因为费用问题而不愿意去医院，那会让病毒传播得更快。

', '

潇湘晨报：当地民众对疫情的看法是什么？

', '

Monica：得克萨斯州的普通民众不知道疫情会变得这么严重。我认为我们的政府夸大其词了，他们并不重视本地疫情。他们认为，只有患有疾病、抵抗力差的老人才会被感染。

', '

美国的疫情不是首先在一个城市爆发。我们有成千上万的美国人在十二月和一月往返飞行，包括我自己。所以每一个人都可能在途中携带感染病毒。

', '

【5】我爱美国也爱中国

', '

潇湘晨报：你会感到害怕吗？

', '

Monica：与其说害怕不如说担忧，关于病毒，关于人们遭受的苦难。我们熬过了西班牙流感和大萧条。我们会挺过去的，但为了恢复到本来的生活，我们必须做出牺牲。

', '

潇湘晨报：什么牺牲？

', '

Monica：数百万人失去了工作，股市下跌，孩子不能重返校园……所有人都身处其中，并受到影响，全世界都在共同面对。

', '

我父亲那一代也受过苦，我相信我们这一代可以承受，因为我们了解苦难。我们也可以教我们的孩子如何面对苦难。

', '

我们遭受痛苦，但并没有那么难熬。对于一个四口之家，你可以得到两份政府补贴，这还不错。如果你找不到工作，你可以失业一年，领取失业救济金，并且不用交税。

', '

对我们大多说人来说，我们可以买个小点的房子，不再买我们不需要的东西。我们这一代的许多人都觉得自己受到了优待，甚至被宠坏了，这对他们来说比较艰难。

', '

潇湘晨报：是否有听说海外华人因疫情遭受到恶意？

', '

Monica：是的，我有听说。我爱中国人民，我相信他们都是勤劳智慧的人，他们在这次疫情中遭受的歧视和恶意是不公平的。我认为偏见是我们最不需要有的事情。

', '

潇湘晨报：你认为美国什么时候能控制住疫情？

', '

Monica：这个不好说，我还是很担忧。你们都回去工作了吗？学校开学了吗？我们都在观察你们的动静看什么时候该回去工作。

', '

潇湘晨报：疫情带给你的思考？

', '

Monica：我爱美国也爱中国。我希望每个人，特别是中国和美国都能从这场疫情中学到些什么。

', '

潇湘晨报：学到什么?

', '

Monica：当灾难发生的时候，不要无视它，尽早接受挑战，采取措施。告诉人们事实和真相，永远和人民站在一起。

', '

潇湘晨报记者 田h
